XGngeo - README

XGngeo is a frontend in GTK 2 wrote in Python for Gngeo, a Neo Geo emulator for Unix...

XGngeo is free software released under the GNU GPL license.
Please see "LICENSE.txt" for more information.

To launch XGngeo, simply type: "python xgngeo.py" in the XGngeo directory.

Full documentation is included in this package in the "doc/" directory.
With that one, you should get XGngeo running without any problem.

For lastest updates and informations about XGngeo, please visit our homepage:
http://choplair.tuxfamily.org/
